🚀 HOW TO USE
═══════════════════════════════════════════════════════════════════════════

1️⃣  INSTALL DEPENDENCIES
    $ pip install -r requirements_prod.txt

2️⃣  RUN COMPLETE PIPELINE

    Option A: Python script (fastest)
    $ python main.py

    Option B: Shell script
    $ chmod +x scripts/run.sh && bash scripts/run.sh

    Option C: Jupyter notebook (interactive)
    $ jupyter notebook train.ipynb
    (Run cells sequentially)

3️⃣  MAKE PREDICTIONS
    $ python scripts/predict.py --data new_data.csv --model bayesian

4️⃣  LOAD MODELS FOR INFERENCE
    from src.models import load_models
    grid_model, bayes_model, scaler = load_models()
    predictions = grid_model.predict(X_scaled)

📊 PIPELINE STAGES
═══════════════════════════════════════════════════════════════════════════

┌─ DATA LOADING (data_loader.py)
│  • Load CSV file
│  • Encode categorical variables
│  • One-hot encoding for multi-class features
│  • Train/test split with stratification (80/20)
│
├─ PREPROCESSING (preprocessing.py)
│  • Outlier removal (IQR method)
│  • SMOTE balancing (training data only - no leakage)
│  • StandardScaler normalization
│
├─ MODEL TRAINING (models.py)
│  • [A] GridSearchCV: 810 parameter combinations
│  • [B] BayesSearchCV: 30 iterations (faster, similar accuracy)
│  • Both optimize Random Forest classifier
│
├─ EVALUATION (evaluation.py)
│  • Accuracy, Precision, Recall, F1-Score
│  • AUC-ROC metrics
│  • Confusion matrices
│  • ROC curves comparison
│
└─ EXPLAINABILITY (explainability.py)
   • SHAP TreeExplainer for feature importance
   • Bar plots (feature ranking)
   • Summary plots (individual contributions)
   • Feature comparison between models

🎯 EXTENDING THE PROJECT
═══════════════════════════════════════════════════════════════════════════

To add new functionality:

1. NEW MODELS?
   → Add to models.py
   → Update main.py to call it

2. NEW PREPROCESSING?
   → Add function to preprocessing.py
   → Update preprocess_pipeline()

3. NEW METRICS?
   → Add to evaluation.py
   → Update evaluate_model()

4. NEW FEATURES?
   → Modify prepare_features() in data_loader.py
   → Update config.py if needed
